セットアップ方法

このページでServiceNow Virtual Agentを動作させるには、ServiceNowインスタンスから提供される埋め込みコードが必要です。

埋め込みコードの設定手順

  1. ServiceNowインスタンスにログイン
  2. Virtual Agent > Designer に移動
  3. 対象のVirtual Agentトピックを選択
  4. 「Portable」タブを開く
  5. 「Website」オプションを選択
  6. 生成されたスクリプトコードをコピー
  7. このHTMLファイルの <!-- SERVICENOW WIDGET CODE HERE --> 部分に貼り付け
