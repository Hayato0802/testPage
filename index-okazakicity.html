Okazaki City - Script Tag 版テストページ

このテストページについて

このページは、ServiceNow Virtual AgentのScript Tag 版が実装されています。

実装状況： dev2okazakicity.service-now.com インスタンスのチャットウィジェットが画面右下に表示されます（CORS設定が完了している場合）。

実装済みコード

このページには既に以下のコードが実装されています（<head>セクション内）： 

<script src="https://dev2okazakicity.service-now.com/scripts/now-requestor-chat-popover-app/now-requestor-chat-popover-app.min.js?sysparm_substitute=false"></script>
<script>
    var chat = new ServiceNowChat({
        instance: 'https://dev2okazakicity.service-now.com/',
        branding: {
            bgColor: '#333333',
            primaryColor: '#0056b3',
            sizeMultiplier: 1.0
        },
        offsetX: 20,
        offsetY: 20,
        position: 'right'
    });
</script>

ポイント： ES6モジュール版（type="module" + import）ではなく、 従来の<script>タグで直接読み込む方式です。

ES6モジュール版との違い

項目 Script Tag 版（このページ） ES6モジュール版（index.html）
読み込み方法 <script src="..."> import ... from "..."
スクリプトURL .min.js（minified） .jsdbx（モジュール）
インスタンス dev2okazakicity blueshipcoltddemo4

注意: チャットウィジェットを表示するには、ServiceNowインスタンス側でCORS設定とPortable Web Clientの有効化が必要です。
