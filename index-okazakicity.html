このテストページについて

このページは、ServiceNow Virtual AgentのES6モジュール版 Portable Web Clientが実装されています。

実装状況： dev2okazakicity.service-now.com インスタンスのチャットウィジェットが画面右下に表示されます（CORS設定が完了している場合）。

実装済みコード

このページには既に以下のコードが実装されています（<head>セクション内）： 

<script type="module">
    import ServiceNowChat from "https://dev2okazakicity.service-now.com/uxasset/externals/now-requestor-chat-popover-app/index.jsdbx?sysparm_substitute=false";

    const chat = new ServiceNowChat({
        instance: "https://dev2okazakicity.service-now.com/",
        context: {
            portal_id: '2d35c60a2bd936104461f77ace91bf91'
        },
    });
</script>

ポイント： type="module" を使用してES6モジュールとしてインポートし、 new ServiceNowChat() でチャットインスタンスを作成します。

注意: チャットウィジェットを表示するには、ServiceNowインスタンス側でCORS設定とPortable Web Clientの有効化が必要です。
