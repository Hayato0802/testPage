外部サイトへの埋め込みコード

以下のコードを外部サイトの <head> セクションに追加してください： 

<script type="module">
    import ServiceNowChat from "https://cdp.city.okazaki.lg.jp/uxasset/externals/now-requestor-chat-popover-app/index.jsdbx?sysparm_substitute=false";
    const chat = new ServiceNowChat({
        instance: "https://cdp.city.okazaki.lg.jp",
        context: {
            portal_id: '2d35c60a2bd936104461f77ace91bf91',
            branding_key: 'now_assist_in_va'
        },
    });
</script>

注意: チャットウィジェットを表示するには、ServiceNowインスタンス側でCORS設定の完了が必要です。
